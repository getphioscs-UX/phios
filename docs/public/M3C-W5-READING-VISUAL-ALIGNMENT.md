# M3C-W5 · Reading Visual Alignment

## Outcome

Reality Reading now presents the frozen Reading result through five explicit
customer views:

1. **Observed Reality** — observable and reported material that the Reading
   carries as evidence;
2. **Runtime Pattern** — the strongest current rule-first pattern, with its
   evidence-threshold status and structural confidence;
3. **Evidence** — the traceable Evidence Trail and the current reliability
   foundation;
4. **Interpretation** — the current possible meaning and any separate
   alternative Reading;
5. **Reading Boundary** — Unknown Reality, limitations, future evidence and
   the existing Navigation-readiness result.

The PHI OS technical model remains available through progressive disclosure.
The header now shows the full seven-stage Runtime Journey.

## Evidence boundary

The customer projection is read-only. It consumes the already generated
Reality Reading and does not:

- call the Reading API or another provider;
- read or write browser storage;
- change an evidence class;
- establish a pattern;
- change the Navigation-readiness decision;
- mutate Runtime, persistence or lineage.

Displayed pattern confidence is the existing structural confidence. It is not
fact probability, diagnosis, prediction or professional certainty.
