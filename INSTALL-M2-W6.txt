PHI OS — M2-W6 Runtime Timeline

1. Copy this package over the repository root and keep the folder structure.
2. Do not delete files that are not included in this package.
3. Run:

   npm run check:runtime-timeline
   npm run check

4. Commit and deploy only after both commands pass.

M2-W6 establishes canonical Runtime Event Types, Event Append Service,
Timeline Reader, customer Timeline Projection, and Event Versioning.
It adds no Runtime Stage and retains the M1 browser Lineage view.
