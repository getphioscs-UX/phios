PHI OS — M2-W5 Runtime Recovery

1. Copy this package over the repository root and keep the folder structure.
2. Do not delete existing files that are not present in this package.
3. Run:

   npm run check:runtime-recovery
   npm run check

4. Commit and deploy only after both commands pass.

M2-W5 adds no Runtime Stage. It adds Recovery Contract, Session Recovery,
Partial Write Recovery, Provider Failure Recovery, and Page Refresh Recovery.
