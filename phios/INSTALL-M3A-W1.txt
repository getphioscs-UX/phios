M3A-W1 Design System Foundation
================================

Required baseline
-----------------
Repository: getphioscs-UX/phios
Branch: main
Commit: 1df74566174ee18457ce8a095fd7a61b14da649f

Install
-------
Extract this ZIP at the repository root and preserve all paths.

Validate
--------
npm run check
node scripts/check-m3a-design-foundation.mjs

Scope
-----
- Expands assets/css/tokens.css.
- Adds Foundation, Typography, Layout, Components and Motion CSS.
- Loads the new layers before legacy page CSS on six baseline pages.
- Supports both legacy .btn and canonical .phi-button.
- Freezes element IDs, translation keys, storage keys, API routes and Runtime
  state boundaries during Design System migration.

----------------
