PHI OS — M2-W1 Registry Layer (reviewed package)

IMPORTANT
This package is based on the current M1 repository state. The three API files
preserve the recent Reading/Navigation language fixes and add only the Registry
startup import. It does not use the stale API versions from the original ZIP.

INSTALL
1. Extract this ZIP.
2. Copy the CONTENTS of M2-W1-Registry-Layer into the root of the phios repo.
3. Allow the listed files and package.json to be replaced.
4. Run:

   npm run check:runtime-registry
   npm run check

✓ M2-W1 Contract, Schema, Version, Migration Registries, M1 parity, and startup validation passed.
