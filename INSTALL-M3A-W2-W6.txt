Install with Visual Studio Code and GitHub Desktop
--------------------------------------------------
1. Confirm GitHub Desktop has pulled the latest main branch.
2. Extract this ZIP.
3. Copy the extracted files into the root of your local phios repository.
4. Allow Windows to merge folders and replace files with the same names.
5. In the VS Code terminal, run:

   npm run check:m3a-public-experience
   npm run check

6. Review the changes in GitHub Desktop.
7. Commit with a message such as:

   M3A-W2-W6 Public Experience

8. Push main and wait for Cloudflare Pages deployment.
